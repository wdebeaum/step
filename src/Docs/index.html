TRIPS Documentation Index

Documentation index for TRIPS source distribution. Some of these items may not exist in a specific configuration, and there may be additional items not catalogued in this index.
  • cvs.html: Instructions for checking out the TRIPS distribution (if you have permission to do this) and a brief introduction to working with CVS.
  • building-trips.html: Basic instructions for building TRIPS from source.
  • running-trips.html: Basic instructions for running TRIPS.
  • trips-devel.html: Some (unattributed) notes on how to run TRIPS when you are developing and debugging. Useful to TRIPS developers.
  • troubleshooting.html: Information on solving common problems, and how to contact the appropriate people in case of uncommon problems with TRIPS.
  • ../config/lisp/README.html: Notes on lisp compatibility for use with TRIPS.
  • CERNL: Instructions relevant to the CERNL project
  • CALO: Documents relevant to the CALO project
  • README-plow.txt: Some instructions for building and installing the PLOW version of TRIPS.
  • README-windows.txt: Building and (eventually) running TRIPS on Windows.
  • making-videos.html: Instructions for making screencast videos of the TRIPS system.

See also the (WIP) trips-docs wiki on github.
