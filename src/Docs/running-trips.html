Running TRIPS

Basic instructions for running TRIPS:

  • Run "$TRIPS_BASE/bin/trips-system", where $TRIPS_BASE is the directory containing the "src", "bin", and "etc" directories, and system is the name of the specific TRIPS system you installed (e.g. cogent).
  • If you have moved the installed system from the configured location, you must specify the new prefix by setting the $TRIPS_BASE environment variable prior to launching the system.
  • A log directory will be created in whatever directory you ran trips from (or you can specify where by setting the environment variable $TRIPS_LOGS prior to running the system). Many people create a tmp or log directory somewhere and then create a symlink from that directory to the command to run TRIPS.

Developers may also be interested in trips-devel.html for alternate ways of running TRIPS during development.

For instructions on building TRIPS from source, see building-trips.html.
